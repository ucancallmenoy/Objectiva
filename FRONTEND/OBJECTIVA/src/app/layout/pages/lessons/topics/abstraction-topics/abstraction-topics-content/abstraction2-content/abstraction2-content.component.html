Abstract Classes

Creating abstract classes: is a class defined using the abstract modifier, and an abstract class cannot be instantiated..

An abstract class is a blueprint that cannot be instantiated but can be extended by other classes.


abstract class Animal {{'{'}}
    abstract void makeSound(); // Abstract Method
    void eat() {{'{'}}
        System.out.println("This animal eats food."); // Concrete method
    {{'}'}}
{{'}'}}

Abstract Methods

An abstract method is a method without a body, meaning it has no implementation. Subclasses must override and provide their own implementation for it.


abstract class Shape {{'{'}}
    abstract void draw(); // No implementation here
{{'}'}}
class Circle extends Shape {{'{'}}
    void draw() {{'{'}}
        System.out.println("Drawing a circle");
    {{'}'}}
{{'}'}}

Concrete Methods in Abstract Class

These methods have a body and can be used as-is by subclasses.

When to Use Abstract Classes

When you want to provide common functionality to subclasses.

When you have shared behavior that subclasses must implement.


// // Abstract class representing a generic Animal
abstract class Animal {{'{'}}
    // Abstract method: All animals make a sound
    abstract void makeSound();

    // Concrete method: All animals eat food
    void eat() {{'{'}}
        System.out.println("This animal eats food.");
    {{'}'}}
{{'}'}}

// Dog class extends Animal
class Dog extends Animal {{'{'}}
    {{'@'}}Override
    void makeSound() {{'{'}}
        System.out.println("Woof Woof");
    {{'}'}}
{{'}'}}

// Cat class extends Animal
class Cat extends Animal {{'{'}}
    {{'@'}}Override
    void makeSound() {{'{'}}
        System.out.println("Meow Meow");
    {{'}'}}
{{'}'}}

// Main class
public class AnimalKingdom {{'{'}}
    public static void main(String[] args) {{'{'}}
        // Create objects of Dog and Cat
        Animal myDog = new Dog();
        Animal myCat = new Cat();

        // Dog's behavior
        System.out.println("Dog:");
        myDog.makeSound();  // Output: Woof Woof
        myDog.eat();       // Output: This animal eats food.

        // Cat's behavior
        System.out.println("\nCat:");
        myCat.makeSound(); // Output: Meow Meow
        myCat.eat();       // Output: This animal eats food.
    {{'}'}}
{{'}'}}
