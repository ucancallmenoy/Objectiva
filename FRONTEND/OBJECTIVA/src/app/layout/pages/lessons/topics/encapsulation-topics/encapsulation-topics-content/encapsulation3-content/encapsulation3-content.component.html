What is the purpose of getters and setters?

Getters and setters are methods that provide controlled access to class properties.

They are fundamental to encapsulation by allowing you to:

Control how values are accessed and modified

Add validation logic

Maintain flexibility to change internal implementation

Support proper encapsulation principles

Naming conventions

Getters and setters are named according to the property they access or modify. The naming convention is to use the word get followed by the property name for getters and set followed by the property name for setters.

Getters



get[PropertyName]();
is[PropertyName]();  // for booleans

Setters


set[PropertyName](TypeValue);

Best practices

When using getters and setters, it's best to:

Always validate input in setters

Keep methods simple and focused

Consider making immutable fields (no setter)

Use meaningful names

Consider lazy initialization in getters

Validation in setters

Validation in setters is the process of ensuring that any data assigned to an object's private fields through setter methods meets specific criteria. This mechanism prevents invalid, inconsistent, or harmful data from being stored in the object's state, thereby maintaining data integrity.

Example: Validating string content if a string value is not empty or null


public void setName(String name) {{'{'}}
  if (name != null && !name.trim().isEmpty()) {{'{'}}
      this.name = name;
  {{'}'}}
  else {{'{'}}
      throw new IllegalArgumentException("Name cannot be empty or null");
  {{'}'}}
{{'}'}}

Real-world examples

Think of getters and setters like a safe deposit box at a bank. The box (private variable) is locked away, but you have special ways to put things in (setters) and take things out (getters).

Here is a simple example using a Person class:


public class Person {{'{'}}
  // Private variables - these are the "hidden" from outside the class
  private String name;
  private int age;

  // Getters for name
  public String getName() {{'{'}}
    return name;
  {{'}'}}

  // Setters for name
  public void setName(String name) {{'{'}}
    this.name = name;
  {{'}'}}

  // Getters for age
  public int getAge() {{'{'}}
    return age;
  {{'}'}}

  // Setters for age with a basic check
  public void setAge(int age) {{'{'}}
    // We can add simple checks in our setters
    if (age => 0 && age <= 120) {{'{'}}
      this.age = age;
    {{'}'}}
    else {{'{'}}  
      throw new IllegalArgumentException("Age must be between 0 and 120");
    {{'}'}}
  {{'}'}}
{{'}'}}

Let's see how to use this class:


public class Main {{'{'}}
  public static void main(String[] args) {{'{'}}
    // Create a new Person
    Person person = new Person();

    // Using setters to give values
    person.setName("John Doe");
    person.setAge(25);

    // Using getters to read values
    System.out.println("Name:" + person.getName()); // Output: Name: John Doe
    System.out.println("Age:" + person.getAge()); // Output: Age: 25

    // Using setters to give invalid values
    person.setAge(150); // Throws an exception: Age must be between 0 and 120
  {{'}'}}
{{'}'}}
