Constructors

It is a special method used to initialize objects. It is called when an object is created and has the same name as the class. It can set initial values for instance variables


public class Student {{'{'}}
    // Default constructor
    public Student() {{'{'}}
        name = "Ako Lang Toh";
        age = 18;
        gpa = 4.0;
    {{'}'}}

    // Parameterized constructor
    public Student(String name, int age, double gpa) {{'{'}}

        this.age = age;
