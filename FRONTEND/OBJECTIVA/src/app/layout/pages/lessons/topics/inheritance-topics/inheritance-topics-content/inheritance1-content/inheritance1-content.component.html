What is Inheritance?

Inheritance is the process by which one object acquires the properties of another object. In object-oriented programming, it allows you to create new classes based on existing ones, enabling code reuse and hierarchy.

A class that is based on another class is said to inherit the other class.

The class being inherited from is known as the parent class, the base class, or the super class.

The class that does the inheriting is called the child class, the derived class, or the subclass.

Few important things about Inheritance:

A derived class automatically inherits all the properties and behaviors of its base class. This allows you to define a base class with shared features when multiple related classes have common characteristics.

A derived class can add features to the base class it inherits defining its own methods and fields. This is one way that a derived class distinguishes itself from its base class.

Types of Inheritance

Single Inheritance

Single inheritance occurs when a class extends another one class. This means the subclass inherits properties and behaviors from a single parent class.

Multilevel Inheritance

Multilevel Inheritance occurs when a child inherits from a parent class, and another child class further inherits from this child.

Hierarchical Inheritance

Hierarchical Inheritance occurs when multiple children inherit from the same parent.

Benefits of Inheritance

Without the use of hierarchies, each object would have to define its own characteristics, leading to redundancy.

However, by use of inheritance, an object only needs to define its unique qualities, inheriting general attributes from its parent class.

Code Reuse

Reuse existing code in new classes.

Extensibility

Easily add or override behaviors.

Maintainability

Centralize and simplify updates.

Real-world examples of inheritance

Inheritance is commonly illustrated using real-world examples, like cars and motorcycles or birds and reptiles.

A Motorcycle is a specialized type of Vehicle class. It inherits attributes like wheels, a driver, and the ability to drive, stop, turn, and crash, while also having unique features like two wheels and handlebars for steering control.
