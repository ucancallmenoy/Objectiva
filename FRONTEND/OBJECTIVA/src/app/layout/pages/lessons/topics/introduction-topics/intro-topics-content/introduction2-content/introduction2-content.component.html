What is class?

Class is a term that describes a group or collection of objects with common properties. It describes what attributes its objects will have and what those objects will be able to do.

In object-oriented programming, a class is like a blueprint that defines how to create objects. It specifies:

  • The data the object will have (called attributes or variables).
  • The actions the object can do (called methods or functions).
